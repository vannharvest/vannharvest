# Knowledge Transfer Document

## Project Overview
This document provides a comprehensive overview of the VannHarvest project, detailing its purpose, objectives, and scope.

## Tech Stack
- **Frontend:** React, Redux
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** AWS, Heroku

## Architecture
The architecture follows a microservices approach, allowing for independent development and scaling of components.

## Components
- **User Interface:** Responsible for user interaction.
- **API Services:** Handles all business logic and data operations.
- **Database Management:** Manages data storage and retrieval.

## Integrations
- **Payment Gateway:** Stripe for processing payments.
- **Authentication:** OAuth 2.0 for user login and registration.

## Deployment
The application is deployed using AWS services and Heroku for CI/CD.
